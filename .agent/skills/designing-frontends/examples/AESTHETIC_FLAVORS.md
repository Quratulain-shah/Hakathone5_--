# Aesthetic Flavors

### 1. Brutalist / Raw
- **Typography**: Heavy grotesque or monospaced fonts.
- **Color**: High contrast, primary colors, or stark black/white.
- **Details**: Thick borders, hard shadows (no blur), system default components styled aggressively.

### 2. Retro-Futuristic
- **Typography**: Geometric sans-serifs or vintage computer fonts.
- **Color**: Cyber-lime, deep navy, or neon accents on dark gray.
- **Details**: Scanlines, CRT flicker effects, glow filters, and grid backgrounds.

### 3. Luxury / Refined
- **Typography**: High-contrast serifs (Didot style) paired with minimal sans-serifs.
- **Color**: Muted earth tones, gold/champagne accents, deep blacks.
- **Details**: Soft fade-ins, generous negative space, subtle grain textures.

### 4. Playful / Toy-like
- **Typography**: Rounded, bubbly fonts.
- **Color**: High-saturation pastels.
- **Details**: Large border-radius, "bouncy" spring animations, glossy/glassmorphism effects.
